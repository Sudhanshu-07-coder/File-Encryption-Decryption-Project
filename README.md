A simple **Python-based File Encryption and Decryption Tool** built using the **Caesar Cipher algorithm**.
The tool allows users to **encrypt and decrypt text files** using a shift key, ensuring that file contents can be protected and restored easily.

This project was developed during my **Python Programming Internship at Codveda Technologies**.

# 🚀 Features

✔ Encrypt text files using Caesar Cipher
✔ Decrypt encrypted files back to original content
✔ User-friendly command line interface
✔ Error handling for missing files
✔ Automatically generates encrypted and decrypted files
✔ Lightweight and easy to run
